CoinC un ensemble de fonctions en C pour faciliter l'utilisation des fonctions arithémtiques de base: Addition, Soustraction, Multiplication, Division, Modulo et Carré.

Comporte 3 fichiers:
--------------------
- Readme.txt : ce fichier, qui présente le projet et les informations de comoilation.
- CoinC.c : les fonctions en C.
- CoinC.h : le fichier include associé.

Pour compiler sous Mac OS X:
----------------------------
- Lancer le terminal.
- Se placer dans le répertoire CoinC et exécuter la commande suivante:
         gcc Coinc.c -o CoinC
